Supporting Economic Transformation Data Portal

This portal aims to support and motivate research focused on structural transformation in developing countries. SET broadly defines economic transformation as a process moving labour and other resources from lower to higher productive activities. This includes moving between sectors towards higher value activities (such as from agriculture to manufacturing) and within sectors (such as from subsistence farming to high-value crops), and usually involves diversification and increased value addition in export activities. Each plot has several interactive features and plot customisation options. Use the interactive tools to the right of the plot to scroll, zoom and box select subportions of the plot. Use the 'Reset' tool to return to the original view and the user can save the plot by using the save option. Where possible. users can download the data directly by clicking the 'Download Data' buttons to the left or below the plot. This exports data in a CSV format which can be read by excel directly.
